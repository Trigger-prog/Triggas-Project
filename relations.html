Relations

A relation is any correspondence between two sets. There are many ways to represent relations.

  • ●Descriptive Rule
  • ●●Mapping Diagram
  • ●Table
  • ●Set of Ordered Pairs
  • ●Equation
  • ●Graph

      • 1. A descriptive rule — is a sentence or expression that tells how to get the output from the input.

      2. Mapping Diagram A mapping diagram shows how each input (x) is paired with an output (y). Usually drawn with arrows connecting values in two ovals.

      3. Table A table lists the inputs and outputs in columns, making it easy to see the pattern.

      4. Set of Ordered Pairs A set of ordered pairs lists the relation as points.

      5. Equation An equation is a mathematical rule written using symbols and variables.

      6. Graph A graph is a picture that shows the relation on the coordinate plane.
