Functions

Now, what is a function? A function is a type of relation or a rule of correspondence between two nonempty sets such that, to each element of the first set called domain, there corresponds one and only one element of the second set called range.
REMARKS: A function is said to be a ‚Äúwell-behaved‚Äù relation. Why? This is because given a starting point, we know exactly where to go. Given a value, say x, we get only and exactly one value of y.
This means that while all functions are relations (since they pair information), not all relations are functions. In fact, functions are a sub-classification of relations.

Given a relation of x and y, we say ‚Äúy is a function of x‚Äù if for every element x in the domain, there corresponds exactly one element y in the range.

GRAPH: FUNCTION OR NOT?

If we are given a graph, how do we know if it‚Äôs a function or not?

It is used to visually determine if a graph represents a function. Just slide any vertical line (think of your pencil) across the graph to see if any two or more points of the graph lie on the same vertical line.
If there are no two or more points on the same vertical line, then the relation is a FUNCTION.
If there are two or more points on the same vertical line then the relation is NOT A FUNCTION.
