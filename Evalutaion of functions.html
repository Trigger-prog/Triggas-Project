FUNCTIONAL NOTATION

Functions are usually given names for easy reference in discussions. The names used are, by convention, the letters of the alphabet.
For example, the function defined by 𝑦 = 𝑥 + 1 may be referred to as 𝒇. We write…
𝑓 𝑥 = 𝑥 + 1
to indicate that the value of the function 𝒇 at x is equal to 𝑥 + 1

FUNCTIONAL NOTATION (cont.)

𝑓 𝑥 DOES NOT mean “𝑓 multiplied by x”. The quantity 𝑓(𝑥) is read as “𝑓 of x” and it is the function value which 𝑓 assigns to x. Hence, f(x) is actually equal to y.
f(x) = y
Here, the set of all possible values of x is the domain and the set of all possible values of y or 𝑓 𝑥 is the range.
Note that you can also use other letters of the alphabet (lowercase or uppercase) in naming a function
